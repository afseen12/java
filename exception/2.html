//2. Write a Java program to create a method that takes an integer as a parameter
and throws an exception if the number is odd.

class OddNumberException extends Throwable{
	public String getMessage(){
		return "Number is odd..";
	}
}

class Main{
	static void checkDigit(int x) throws OddNumberException {
		
		if(x % 2 != 0){
			throw new OddNumberException();
		}			
	}
public static void main(String st[]){
	
	try{
	checkDigit(9);
	}
	catch(OddNumberException ex){
		System.out.println(ex.getMessage());
	}
}
}

Output:

Number is odd..
