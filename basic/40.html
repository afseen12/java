//Java program to find surface area of cube

    class SurfaceAreaOfCube{
        double side;
        SurfaceAreaOfCube(double s){
            side = s;
        }
        double surfaceAreaOfCube(){
            return 6*side*side;
        }
    }
    
    class Main{
        public static void main(String st[]){
        SurfaceAreaOfCube cube = new SurfaceAreaOfCube(3);
        System.out.println("Surface area of cube : "+cube.surfaceAreaOfCube());
        }
    }

Output:

Surface area of cube : 54.0
