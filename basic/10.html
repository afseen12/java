//Write a Java program and compute the sum of the digits of an integer.

    class SumOfDigits{
        public static void main(String st[]){
            int num = 786;
            System.out.println("Number : " +num);
            int sum = 0;
            while(num!=0){
                sum += num%10;
                num /= 10;
            }
        
        System.out.println("Sum of digits : " +sum);
        }
    }

Output:

Number : 786
Sum of digits : 21
