//9. How to convert String to Date

    class StringToDate{
        public static void main(String st[]) throws java.text.ParseException
        {
            String str = "2024-01-01";
            java.time.LocalDate date = java.time.LocalDate.parse(str);
            System.out.println("Converted Date : "+date);
        }
    }

Output:

Converted Date : 2024-01-01
