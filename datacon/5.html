//5. How to convert String to float

    class StringToFloat{
        public static void main(String sy[]){
            String str = "1.23";
            try{
                float f = Float.parseFloat(str);
                System.out.println("Converted float : "+f);
            }
            catch(NumberFormatException ex){
                System.out.println(ex.getMessage());
            }
        }
    }

Output:

Converted float : 1.23
