/*15. Write a Java program to create an abstract class Animal with an 
abstract method called sound(). Create subclasses Lion and Tiger that 
extend the Animal class and implement the sound() method to make a 
specific sound for each animal.*/

    abstract class Animal{
        abstract void sound();
    }
    
    class Lion extends Animal{
        void sound(){
            System.out.println("Lion Roar");
        }
        
    }
    
    class Tiger extends Animal{
        void sound(){
            System.out.println("Tiger Growl");
        }
    }
    
    class Main{
        public static void main(String args[]){
            Lion obj = new Lion();
            obj.sound();
            Tiger obj1 = new Tiger();
            obj1.sound();
        }
    }

Output:

Lion Roar
Tiger Growl
