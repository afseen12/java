/*7. Write a Java program to create a class known as Person with methods 
    called getFirstName() and getLastName(). Create a subclass called
    Employee that adds a new method named getEmployeeId() and overrides 
    the getLastName() method to include the employee's job title.*/

    class Person{
    String fname, lname;
        Person(String F, String L){
            fname = F;
            lname = L;
        }
        String getFirstName(){
            return fname;
        }
        String getLastName(){
            return lname;
        }
    
    }
    
    class Employee extends Person{
    String Empid, Jtitle;
        Employee(String F, String L, String E, String J){
            super(F, L);
            Empid = E;
            Jtitle = J;
        }
        String getEmployeeID(){
            return Empid;
        }
        String getLastName(){
            return super.getLastName() + " , "+ Jtitle;
        }
    }
    
class Main{
    public static void main(String args[]){
    Employee obj = new Employee("Misbah", "Ansari", "E123", "Engineer");	
    System.out.println("The First Name is "+obj.getFirstName());
    System.out.println("The Last Name and Employee's Job Title is "+obj.getLastName()+"respectively.");
    System.out.println("The Employee ID is "+obj.getEmployeeID());
        
        }
    }

Output:

The First Name is Misbah
The Last Name and Employee's Job Title is Ansari , Engineer respectively.
The Employee ID is E123
