/*2. Write a Java program to create a class called Vehicle 
    with a method called drive().Create a subclass called Car that overrides 
    the drive() method to print "Repairing a car".*/

    class Vehicle{
        void drive(){
            }
    }
        
    class Car extends Vehicle{
        void drive(){
            System.out.println("Repairing a car");
        }
    }
        
    class Main{
        public static void main(String args[]){
        Car obj = new Car();
        obj.drive();
        }
    }


Output:

Repairing a car
