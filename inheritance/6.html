/*6. Write a Java program to create a class called Animal with a method
    named move(). Create a subclass called Cheetah that overrides the move()
    method to run.*/

    class Animal{
        void move(){
            System.out.println("Animals Run fast");
        }
    }
    
    class Cheetah extends Animal{
        void move(){
            System.out.println("Run");
        }
    }
    
    class Main{
        public static void main(String args[]){
            Cheetah obj = new Cheetah();
            obj.move();
        }
    }

Output:

Run
